⭐ Focus on Core Backend Logic
The main goal of this project was implementing all essential backend functionalities of a real online store using Django.

🗂️ Product Management System
Includes product listing, detail pages, categories, and dynamic data handling.

🛍️ Shopping Cart Functionality
Full add/remove/update cart system with session-based cart management.

📦 Order Creation & Checkout Flow
Users can place orders, confirm details, and complete the checkout process.
Backend logic for order creation, validation, and saving is fully implemented.

👤 User Authentication & Profiles
Basic login/register system plus user-specific order history.

💳 Price Calculations
Accurate backend calculation for totals, items, and ...

🧩 Clean and Modular Backend Structure
Each core feature is separated into clean Django apps for scalability.
Backend structure is Django MVT.
